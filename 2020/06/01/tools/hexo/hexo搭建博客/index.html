mac hexo+github 搭建博客

  3. 3. 指定目录 并初始化

install node.js

brew install node / https://nodejs.org/en/download/

install hexo

npm install -g hexo-cli

指定目录 并初始化

mkdir /Users/**/gitblog/cont
hexo init
(可能出现问题，找不到或无法加载主类： nmp install; hexo s )

本地运行

hexo s(打开localhost:4040,查看效果)

部署到github

(1)
新建仓库 xyz.github.io（xyz为github用户名）

(2)
修改目录配置文件 _congif.yml(xyz替换为github用户名)

deploy:
type: git
repo: https://github.com/xyz/xyz.github.io.git
branch: master

(3)
npm install hexo-deployer-git –save
hexo clean
hexo deploy

(4)
https://xyz.github.io/

发布一篇博客

(1)
hexo new ‘HBase概览’ -p nosql/hbase/HBase概览

(2)
source/_posts/nosql/hbase/目录下找到刚才建立的 HBase概览.md文件, 进行编辑
title: Hello post #文章标题
date: 2014-08-05 11:15:00 #发表时间
categories: #分类
toc: true
tags: #标签，多个标签时可以用[标签1,标签2]的方式，或者”- 标签“的方式每行一个

(3)
编辑完文章后依次执行hexo g和hexo s,然后访问localhost:4000来预览效果

(4)
没有问题了以后，执行hexo d来同步到GitHub
